# 關於 TC - Protfolio

本篇收錄 2023 年到 2024 年間製作，最接近近期狀態的幾個網頁設計作品。  
其中各別標注設計專案時使用的核心技能，  
並附加 GitHub Repo 與 Demo site 連結供參考。

# 使用技術

#React #vite #Bootstrap

# 環境版本
Node - v18  
Npm - v9  
Bootstrap - v5  
React - v18  
Vite - v5

# 資料夾結構

- src # main.jsx, App.jsx, index.css, 以及 mockup.html 放置處  
  - assets # 圖片放置處  
src/assets/scss # SCSS 樣式放置處
